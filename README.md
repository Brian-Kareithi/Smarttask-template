# SmartTask

A full-stack task management and productivity application built with Flutter & Firebase.

## Features

- **Authentication**: Email/password sign up & login with Firebase Auth
- **Dashboard**: Progress overview, stats grid, analytics charts (line + pie), recent tasks
- **Task Management**: Full CRUD with real-time Firestore sync, search, filter tabs, priority & category badges
- **Collaboration**: Add/remove collaborators by email on any task
- **Dark Mode**: Toggleable dark/light theme, persisted to Firestore
- **Profile**: Profile card with stats, settings toggles, edit profile, help modals

## Tech Stack

| Layer | Choice |
|---|---|
| Framework | Flutter (>=3.29, Dart 3.x) |
| Backend | Firebase (Auth, Firestore, Storage) |
| State Management | Provider |
| Navigation | GoRouter + custom pill-style bottom tabs |
| Charts | fl_chart |
| Animations | Custom animated liquid blob, fade/slide transitions, staggered list |

## Firebase Setup

```bash
# Install Firebase CLI and activate FlutterFire
dart pub global activate flutterfire_cli

# Configure Firebase for this project
flutterfire configure --project=smarttask-baff9
```

Then run `flutterfire configure` to regenerate `lib/firebase_options.dart` with the correct platform configs.

## Getting Started

```bash
# Install dependencies
flutter pub get

# Run on device/emulator
flutter run

# Run on web
flutter run -d chrome
```

### Windows Setup

If you encounter a symlink error (`Building with plugins requires symlink support`):

1. Enable **Developer Mode** in Windows Settings:
   - Run `start ms-settings:developers`
   - Toggle **Developer Mode** ON
2. Re-run `flutter pub get`
3. Run the app again
